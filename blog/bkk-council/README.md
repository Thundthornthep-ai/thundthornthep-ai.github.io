## <a id="phases"></a>4. Phase A-E คู่มือ 5 ระยะ

### Phase A: ก่อนเลือกตั้ง
- คุณสมบัติและลักษณะต้องห้ามของผู้สมัคร ส.ก.
- กระบวนการประกาศเลือกตั้งและรับสมัคร
- เงื่อนไขการเลือกตั้ง (ครบวาระ 60 วัน / ยุบสภา 60 วัน / ตำแหน่งว่าง 90 วัน)
- วันจับเบอร์ — มาตรา 54-55

### Phase B: หาเสียง
- DO / DON'T ของการหาเสียง
- กฎเกณฑ์ป้ายหาเสียง (ม.60-62 + ระเบียบ กกต.)
- **ค่าใช้จ่าย 12 ประเภท** ตามประกาศ กกต. 2563 ข้อ 3(1)-(12)
- วันยุติการหาเสียง + วันเลือกตั้ง

### Phase C: หลังเลือกตั้ง
- ประกาศผลการเลือกตั้ง (ม.106, 109)
- ระบบใบสี (เขียว / เหลือง / ส้ม / แดง)
- การยื่นบัญชีรายรับ-รายจ่าย ภายใน 90 วัน (ม.62)
- การยื่นทรัพย์สิน ป.ป.ช.

### Phase D: ปฏิบัติหน้าที่
- อำนาจหน้าที่ ส.ก. 27 ด้าน
- ข้อบังคับการประชุมสภา กทม.
- กรรมการสามัญ 13 คณะ
- การพิจารณาข้อบัญญัติ + งบประมาณ
- กระทู้ถาม + ญัตติ

### Phase E: คลังข้อมูล
- สิทธิประโยชน์ ส.ก. (เงินเดือน, เบี้ยประชุม, สวัสดิการ)
- การสิ้นสภาพ + การเลือกตั้งใหม่
- เอกสาร Timeline ทั้งหมด

## <a id="campaign-tracker"></a>5. Campaign Tracker (`campaign-tracker.html`)

ระบบบัญชีค่าใช้จ่ายเลือกตั้ง — ทำงานบน browser ไม่ต้องติดตั้ง

### Features

| Tab | หน้าที่ |
|-----|---------|
| **Dashboard** | วงเงิน กกต. vs ใช้ไป + progress bar + แจ้งเตือน 90% |
| **ทีมงาน** | เพิ่ม/ลบ ผู้ช่วยหาเสียง (ชื่อ, หน้าที่, ค่าตอบแทน/วัน) |
| **รถหาเสียง** | เพิ่ม/ลบ (ทะเบียน, ประเภท, ค่าเช่า/วัน) |
| **ป้ายหาเสียง** | เพิ่ม/ลบ (ประเภท, จำนวน, สถานที่, ราคา) |
| **12 ประเภท** | รายละเอียด 12 ประเภทค่าใช้จ่ายตามกฎหมาย |
| **รายรับ-รายจ่าย** | บันทึกรายการ + สรุปแยกหมวด |
| **พิมพ์ส่ง กกต.** | พิมพ์รายงาน ตามแบบ ส.ถ./ผ.ถ. 1/13 |

### วงเงินตามเขต (ประกาศ กกต. เลือกตั้ง 2565)
| วงเงิน | เขต |
|---------|-----|
| 820,000 | สัมพันธวงศ์, ป้อมปราบฯ, บางรัก, ปทุมวัน, พระนคร |
| 900,000 | ราชเทวี, พญาไท, วัฒนา, คลองเตย, สาทร, บางคอแหลม, บึงกุ่ม, ทุ่งครุ, ดุสิต, ทวีวัฒนา |
| 1,000,000 | บางพลัด, สะพานสูง, ลาดพร้าว, ธนบุรี, คลองสาน, จอมทอง, ห้วยขวาง, ยานนาวา, ตลิ่งชัน, สวนหลวง, วังทองหลาง |
| 1,050,000 | ภาษีเจริญ, บางซื่อ, จตุจักร, หนองแขม, พระโขนง, บางนา, ราษฎร์บูรณะ, ดินแดง, คันนายาว, บางบอน |
| 1,150,000 | บางเขน, บางแค, ดอนเมือง, สายไหม, มีนบุรี, หนองจอก, ลาดกระบัง, บางกอกน้อย, หลักสี่, บางขุนเทียน |

> วงเงินจากประกาศ กกต. เลือกตั้ง 2565 — รอประกาศใหม่สำหรับเลือกตั้ง 2569

### การจัดเก็บข้อมูล
- ข้อมูลเก็บใน **localStorage** ของ browser (prefix `sk_`)
- ปิดเปิด browser → ข้อมูลยังอยู่
- **Export JSON** → สำรองข้อมูลเก็บไว้
- **Import JSON** → กู้คืนข้อมูลจากไฟล์สำรอง
- **Export CSV** → เปิดใน Excel ได้

### ข้อจำกัด
- ข้อมูลอยู่ใน browser เครื่องเดียว (ไม่ sync ข้ามเครื่อง)
- หาก clear browser data → ข้อมูลหาย (ใช้ Export สำรองก่อน)
- รายงานพิมพ์เป็น **draft** — ตรวจสอบกับแบบฟอร์ม กกต. จริงก่อนยื่น

## <a id="district-intel"></a>6. District Intelligence (`district-intel.html`)

GREEN FLAG Tool — ข้อมูลพื้นที่ 50 เขต กทม.

### Features
- **เลือกเขต** (dropdown 50 เขต) → แสดงข้อมูลเฉพาะเขต
- **POI สถานที่สำคัญ** — วัด, ชุมชน, โรงเรียน, ตลาด, คอนโด, สถานพยาบาล ฯลฯ
  - Hardcode ข้อมูลเริ่มต้น + ผู้ใช้เพิ่มเองได้
- **ปักธง / Check-in** — บันทึกสถานที่ที่ไปแล้ว + ปัญหาที่พบ + ความต้องการชุมชน
- **Traffy Fondue** — ลิงก์ตรงไปหน้า Traffy ของแต่ละเขต
- **เว็บสำนักงานเขต** — ลิงก์ตรงไปเว็บ 50 เขต
- **Export JSON** — สำรองข้อมูล check-in ทั้งหมด

### การใช้งาน
1. เลือกเขตจาก dropdown
2. ดูสถานที่สำคัญ + เพิ่ม POI ของตัวเอง
3. กด "ปักธง" เพื่อบันทึกการ check-in
4. กด "Traffy Fondue" เพื่อดูปัญหาจากประชาชน
5. Export ข้อมูลเป็น JSON เพื่อสำรอง

## <a id="downloads"></a>7. คลังเอกสาร PDF (32 ไฟล์)

เข้าถึงได้ผ่าน Phase E (คลังข้อมูล) และลิงก์ในแต่ละ Phase

- พ.ร.บ.เลือกตั้งท้องถิ่น 2562
- ฉบับที่ 2 (2566) แก้ไขเพิ่มเติม
- พ.ร.บ.ระเบียบบริหาร กทม. 2528
- พ.ร.ป. ป.ป.ช. 2561

### แบบฟอร์ม + คู่มือ (6 ไฟล์)
- ใบสมัคร ส.ถ. 4/1
- แบบบัญชีรายรับรายจ่ายเลือกตั้ง
- คู่มือการรับสมัคร + ขั้นตอน
- ชุดความรู้เลือกตั้ง ส.ก. 65
- ความผิดเกี่ยวกับการเลือกตั้ง

### ข้อบังคับ / จรรยาบรรณ สภา กทม. (8 ไฟล์)
- ข้อบังคับการประชุมสภา กทม. 2562 + แก้ไขครั้งที่ 2-4
- จรรยาบรรณสมาชิกสภา กทม. 2559
- ระเบียบรับเรื่องร้องเรียน + กระทู้ถามสด
- อำนาจหน้าที่สภา กทม.

### คู่มือปฏิบัติงาน + สรุป (8 ไฟล์)
- คู่มือสมาชิกสภา กทม. เล่ม 1-2
- สรุปรายตอน 1-6 (อำนาจหน้าที่, การประชุม, กรรมการ, ญัตติ, การรับรอง, กระทู้ถาม)

### ระเบียบหาเสียง + อื่นๆ (6 ไฟล์)
- ระเบียบวิธีหาเสียง 2563 + ฉบับแก้ไข
- พ.ร.ฎ. เงินเดือน ส.ก. 2560
- ประมวลจริยธรรมท้องถิ่น 2565

## <a id="tech"></a>8. เทคโนโลยี

| เทคโนโลยี | ใช้สำหรับ |
|-----------|----------|
| Static HTML/CSS/JS | เว็บทั้งหมด — GitHub Pages |
